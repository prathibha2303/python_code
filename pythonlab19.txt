1. Calculate the total revenue generated by two product categories in a store
Input:
category1_revenue = np.array([500, 600, 700, 550])
category2_revenue = np.array([450, 700, 800, 600])
Output:
Total Revenue: [ 950 1300 1500 1150]

Code:
import numpy as np											#importing numpy
category1_revenue = np.array([500, 600, 700, 550])							#creating an array
category2_revenue = np.array([450, 700, 800, 600])							#creating an array
result=category1_revenue +category2_revenue								#adding two array
print("the total revenue is:",result)									#printing the result


Output:
the total revenue is: [ 950 1300 1500 1150]

2. Calculate the profit made by a company 

Input: 
revenue = np.array([10000, 12000, 11000, 10500]) 
expenses = np.array([4000, 5000, 4500, 4800]) 

Output: 
Profit: [6000 7000 6500 5700]

Code:
import numpy as np												#importing numpy
revenue = np.array([10000, 12000, 11000, 10500]) 								#creating an array in numpy
expenses = np.array([4000, 5000, 4500, 4800]) 									#creating an array in numpy
result=revenue-expenses											#doing substraction
print("the profit made by a company is",result)									#printing the result

Output:
the profit made by a company is [6000 7000 6500 5700]

3. Determine which products in a store are out of stock (quantity is 0). 

Input:
inventory = np.array([10, 0, 5, 0, 20, 0]) 

Output: 
Out of Stock Products: [0 0 0]

Code:
import numpy as np											#importing an numpy
inventory = np.array([10, 0, 5, 0, 20, 0]) 								#creating an array in numpy
out_of_stock=inventory==0										#generating a boolean array to identify out-of-stock items (where quantity is 0)
result=inventory[out_of_stock]										# used to filter out and get the out of stock products
print(result)											#print statements

Output:
[0 0 0]

4.Calculate the total cost of items in a shopping cart, considering the quantity and price per item.

 Input: 
quantity = np.array([2, 3, 4, 1]) 
price_per_item = np.array([10.0, 5.0, 8.0, 12.0]) 

Output: 
Total Cost of Items: [20. 15. 32. 12.]

Code:
import numpy as np											#importing numpy
quantity = np.array([2, 3, 4, 1]) 									#defining array
price_per_item = np.array([10.0, 5.0, 8.0, 12.0]) 								#creating an array
result=quantity*price_per_item									#doing product 
print(result)											#print the statements

Output:
[20. 15. 32. 12.]
